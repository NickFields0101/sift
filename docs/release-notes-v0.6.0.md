# SIFT v0.6.0

Version 0.6.0 turns SIFT into a fuller local Xahau/Evernode venture workspace while preserving the deterministic review boundary.

## Highlights

- New **Research & Run** flow: generate or select an idea, draft a shadow evaluation, search public sources through OpenRouter, validate provider-returned citation excerpts, and calculate a provisional outcome with one consolidated source approval.
- Public web findings are permanently capped at `DeskResearch / E1`; AI cannot invent customer interviews, commitments, payments, audits, production behavior, or reviewer verification.
- Citation-only provenance: model-authored URLs are ignored, excerpts must match provider annotation content exactly, and the ledger retains source URL, title, retrieval time, excerpt hash, model, and search provider.
- Atomic approval: the live project remains unchanged until the final packet confirmation; malformed or stale sources fail the entire batch without partial writes.
- New guarded **Build** workspace for Evernode MCP, Xahau MCP, XAHC, and XAHC Prover with fixed, unsigned, allowlisted operations.
- Complete SIFT visual identity, updated tornado assets, polished light/dark themes, responsive layouts, and friendlier OpenRouter model search.
- Windows installer and portable packages use the SIFT product name. macOS Intel and Apple-silicon packages are built through the release workflow.

## Upgrading to SIFT

- Existing desktop projects and protected model settings remain available. SIFT keeps the established installer, bundle, and user-data identities internally so an upgrade does not create a second Windows installation or abandon encrypted credentials.
- Browser project data and unfinished personality drafts move automatically to SIFT storage keys on first use.
- On macOS, drag `SIFT.app` into Applications. After confirming the new app opens and your project is present, remove the pre-SIFT app bundle if it is still shown separately. Do not delete its Application Support data.
- The release files, shortcuts, window titles, executable metadata, and all current documentation now use the SIFT name.

## Important boundaries

- Research & Run currently requires a user-supplied OpenRouter API key and credits. Model-token and web-search charges apply.
- OpenRouter web search is a beta provider feature. SIFT fails closed when attributable citation excerpts are unavailable.
- A researched outcome is still provisional. E1 public research does not replace direct customer evidence, human gate decisions, independent security review, or a final investment/launch decision.
- SIFT does not sign or submit transactions, accept wallet secrets, acquire leases, spend funds, or deploy contracts automatically.
